Q-1 What is Redux?


Redux is a state management library commonly used with JavaScript applications, especially those built with React. It helps manage the state of an application in a predictable way, making it easier to understand and debug complex applications.

=>>Benefits of Using Redux:

1)Predictability:
Since the state is managed in a single place and can only be changed through actions, the flow of data becomes predictable and easier to trace.

2)Debugging:
Redux DevTools allows developers to track state changes over time, making it easier to debug applications.

3)Consistency:
By managing the state in a central store, Redux helps ensure that the application's state is consistent across different components.
